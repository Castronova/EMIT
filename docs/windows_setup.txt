Prequisites

* Install Visual C++ 2008 Express Edition (required for compiling via pip)
    Find vsvarsall.bat (e.g. C:\Program Files (x86)\Microsoft Visual Studio 14.0\VC).  Doubleclick on vcvarsall.bat to load environment variables, then reopen cmd prompt.
    `http://stackoverflow.com/questions/4676728/value-error-trying-to-install-python-for-windows-extensions`
